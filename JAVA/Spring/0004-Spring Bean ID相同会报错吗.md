# Spring Bean ID相同会报错吗

| 配置场景              | 是否报错   | 最终结果                                   |
|-------------------|--------|----------------------------------------|
| XML (同文件)         | 是      | 启动失败，抛出解析异常。                           |
| XML (不同文件)        | 否 (默认) | 后者覆盖前者，只保留后加载的 Bean。                   |
| Java Config (同类)  | 否      | 只注册第一个，后面的被忽略（极易引发 Bug）。               |
| Java Config (不同类) | 是 (默认) | 抛出 ConflictingBeanDefinitionException。 |
