# 过滤器和拦截器有什么区别

| 对比维度 | 过滤器 (Filter)                         | 拦截器 (Interceptor)                                   |
|------|--------------------------------------|-----------------------------------------------------|
| 所属规范 | Java Servlet 规范 (Jakarta EE)         | Spring MVC 框架，Spring MVC 框架层 (DispatcherServlet 内部) |
| 拦截范围 | Web 容器层 (如 Tomcat)，所有请求，包括静态资源、JSP 等 | 仅限由 Spring MVC 处理的请求 (Controller)                   |
| 依赖注入 | 默认不支持，需特殊配置                          | 原生支持，可直接注入 Spring Bean                              |
| 核心方法 | doFilter()                           | preHandle(), postHandle(), afterCompletion()        |
